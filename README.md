### What I learned

- How to fork a repository on GitHub.
- Tag class shortcut.
  >`tag.classname#idname` + `tab`
- Change occurences on siblings.
  >`cmd` + `d`
- How to use a CSS reset.
  >```css
  >* {
  >  padding: 0;
  >  margin: 0;
  >  box-sizing: border-box;
  >  font-family: Helvetica, sans-serif;
  >}
  >```
